# 📸 SmartCameraAI

Ứng dụng Android hỗ trợ nhận diện hình ảnh thông minh bằng camera và ảnh từ thư viện, kết hợp AI để phân tích và hiển thị kết quả trực quan.  
Dự án được phát triển bằng Android Studio cùng Firebase Authentication và các công nghệ xử lý ảnh hiện đại.

---

# 🚀 Chức năng chính

- 🔐 Đăng ký / Đăng nhập tài khoản
- 🏠 Giao diện Home trực quan
- 🤖 Nhận diện đối tượng bằng Camera
- 🖼️ Nhận diện ảnh từ thư viện
- 🕘 Lưu lịch sử quét
- 👤 Quản lý thông tin cá nhân
- ✏️ Chỉnh sửa hồ sơ cá nhân
- 🔑 Đổi mật khẩu
- 💬 Trung tâm hỗ trợ người dùng

---

# 🛠️ Công nghệ sử dụng

| Công nghệ | Mô tả |
|---|---|
| Java | Ngôn ngữ lập trình chính |
| Android Studio | Môi trường phát triển |
| Firebase Authentication | Xác thực người dùng |
| Firebase Realtime Database / Firestore | Lưu dữ liệu |
| CameraX | Quét và nhận diện bằng camera |
| AI Image Recognition | Nhận diện hình ảnh |
| RecyclerView | Hiển thị lịch sử quét |
